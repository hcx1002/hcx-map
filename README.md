# 腾讯地图VUE组件

## 获取地图经纬度，搜索地图位置

#### 本工具方便后台管理系统获取地图标点经纬度，支持搜索地点，地点列表

## 安装

```
npm install
```

### 运行

```
npm run serve
```

### 构建

```
npm run build
```
